# Welcome to Metis Design System

A React component library built with TypeScript, Tailwind CSS, and Radix UI.

## Installation

```bash
npm install @metis-w/design-system
```

## Setup

Add the design system styles to your project's main CSS file (usually `index.css` or `globals.css`):

```css
@import "@metis-w/design-system/styles";
```

## Usage

```tsx
import { Button } from "@metis-w/design-system";

function App() {
    return <Button variant="accent">click me</Button>;
}
```
